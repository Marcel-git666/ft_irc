# 1. Description

## Project Overview
This project is a custom **IRC (Internet Relay Chat) server** developed as part of the 42 curriculum.
Project was developed based on **Weechat** client, but it accepts allowing multiple users to join channels and exchange messages in real-time.

**Key Features:**
- Multiple client support
- Channels (creation, joining, leaving)
- Private messaging
- Robust error handling and server validation
